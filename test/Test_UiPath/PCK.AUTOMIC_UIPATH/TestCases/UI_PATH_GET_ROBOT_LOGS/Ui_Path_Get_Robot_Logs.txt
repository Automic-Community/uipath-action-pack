Get Robot Logs Template
    [Arguments]    ${OK_NOK}    ${RobotName}    ${Properties}    ${Filter}    ${OrderBy}    ${Top}
    ...    ${Skip}    ${Include_Count}    ${Expand}
    [Documentation]    This test is to test the feature of Get_Robot_Logs .
    ...    -${OK_NOK}: Boolean value indicate if the action is ENDED_OK or ENDED_NOT_OK. Value of this variable should be OK or NOK.
    ...    -${RobotName}: This value specifies the name of robot.
    ...    -${Properties}: This field specifies the list of properties as comma separated to include in the response. E.g. ProcessName,TimeStamp.
    ...    -${Filter}: This field specifies the filters to be applied for the results, based on a Boolean condition. E.g. ProcessName eq 'SalesProcess.
    ...    -${OrderBy}: This field specifies specify the property to be used in ordering. E.g. TimeStamp desc
    ...    -${Top}: This field specifies if the result should include only the first N items.
    ...    -${Include_Count}: This field specifies the count user wants to includes.
    ...    -${Skip}: This field specifies if the result should skip the first N items.
    ...    -${Expand}: This field specifies related entities to be inlined..
    Generate String
    Init Action    ${_GET_ROBOT_LOGS_ACTION}
    Connect To UiPath
    Action Set    &UC4RB_UIP_ROBOT_NAME#    ${RobotName}
    Run Keyword If    '${Properties}' == 'Invalid'    Action Set    &UC4RB_UIP_PROP#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_PROP#    ${Properties}
    Run Keyword If    '${Filter}' == 'Invalid'    Action Set    &UC4RB_UIP_FILTER#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_FILTER#    ${Filter}
    Run Keyword If    '${OrderBy}' == 'Invalid'    Action Set    &UC4RB_UIP_ORD_BY#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_ORD_BY#    ${OrderBy}
    Run Keyword If    '${Top}' == 'Invalid'    Action Set    &UC4RB_UIP_TOP#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_TOP#    ${Top}
    Run Keyword If    '${Skip}' == 'Invalid'    Action Set    &UC4RB_UIP_SKIP#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_SKIP#    ${Skip}
    Run Keyword If    '${Include_Count}'    Action Set    &UC4RB_UIP_INC_CNT#    ${Include_Count}
    Run Keyword If    '${Expand}' == 'Invalid'    Action Set    &UC4RB_UIP_EXPAND#    ${Generated_Name}_${Time}    ELSE    Action Set
    ...    &UC4RB_UIP_EXPAND#    ${Expand}
    Action Execute
    Run Keyword If    '${OK_NOK}' == 'OK'    Assert Success    ELSE    Assert Failure
